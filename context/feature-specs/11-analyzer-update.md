We create the functionality to get the updates for eac analyzer

## Create the api route

Create the api route to hold the functionality

- Create an api route `api/updates/route` to hold the functionality for getting the updates
- the api will be called from the get updates button of each result in te analyzer result
- Create the functionality using the cypto compare api
- when the api body or search params is sent you are to spilt the string by the comma
- Then use the cypto compare api to get the updates for that specific params
- Get the api key from the .env file
- log the reply to the console

Do not attach any other functionalities or UI
